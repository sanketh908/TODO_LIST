This To-Do List application is a simple web application built using Spring Boot, Spring MVC, and Thymeleaf that allows users to create, view, update, and delete tasks. Users can add new tasks, see a list of all existing tasks, mark tasks as completed or not completed, and delete tasks when they are no longer needed. The application follows a clean MVC architecture, separating responsibilities between Controller, Service, and Repository layers. It is developed using Java, Spring Boot, Spring MVC, Thymeleaf, Spring Data JPA, Hibernate, and Maven, and it supports either MySQL or H2 database depending on the configuration. The main endpoints include GET “/” to view all tasks, POST “/” to create a new task, GET “/{id}/toggle” to toggle a task’s completion status, and GET “/{id}/delete” to delete a task.
